# ⚡ Arduino Reaction Game

My first Arduino project — a simple and fun two-player reaction game built using LEDs, push buttons, and a buzzer.

## 🎮 How It Works

- The Green, Yellow, and Red LEDs blink in sequence.
- A buzzer sounds along with the LED sequence.
- After a random amount of time, the sequence stops.
- The Blue LEDs turn ON, signaling players to react.
- The first player to press their button wins the round.

## 🛠 Components Used

- Arduino Uno
- 1 × Green LED
- 1 × Yellow LED
- 1 × Red LED
- 2 × Blue LEDs
- 2 × Push Buttons
- Buzzer
- Breadboard
- Jumper Wires
- Resistors
